 в языке go  есть только основные типы переменных как и в других яп, а именно:
 int
 - float (32|64)
-  rune 
 - string 
 - bool
 объявление переменных может происходить 3 методами:
 
 - **явный объявление с типом:
 ```
 var x int 
 x=10
 ```
 >используется в случаях, когда мы не знаем какие данные будут помещены в переменную
 >Также при  объявлении таким типом и не передавая значения в переменную переменной автоматически выдается значение по умолчанию (zero values)
 ```
 var i int // 0
 var s string // пустая строка ""
 var b bool //false
 ```
 
- **объявление и инициализация сразу:
``` 
  var msg string = "hello" // для одной переменной
  var a, b int =1,2 // для нескольких
  ```
^[почти не используется на практике

- **краткая запись (чаще всего используется):
```
x:=10 // для одной переменной
a,b:= 2,"go" // для нескольких переменных
   ```
^[тип присваивается автоматический в зависимости от переданных данных

также существует способ записывания переменных  в многострочном варианте:
```
var (
	i=10
	s="shi"
)
```
^[Удобно для группировки логически связанных переменных]

по типу объявления данные делят на:
- **Глобальные** - объявленные вне функций, доступна в любом месте проекта
>используется редко только для переменных неизменяемых и нужных во всем проекте по типу числа pi или же константного токена

- **Локальные** - объявленные внутри функций
>локальные переменные могут изолироваться, не давая путаться в проекте
